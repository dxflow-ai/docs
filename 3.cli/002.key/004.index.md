---
title: Key
description: Manage authentication keys for engine access
---

### Usage

```bash [Terminal]
dxflow key
```

### Subcommands

| Command | Description | Help |
|---------|-------------|------|
| [**list**](/docs/cli/key/list) | List auth keys | `dxflow key list --help` |
| [**register**](/docs/cli/key/register) | Add auth key | `dxflow key register --help` |
| [**unregister**](/docs/cli/key/unregister) | Remove auth key | `dxflow key unregister --help` |
| [**generate**](/docs/cli/key/generate) | Create key pair | `dxflow key generate --help` |
