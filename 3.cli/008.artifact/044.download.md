---
title: Artifact Download
description: Download artifact from remote storage
---

### Usage

```bash [Terminal]
dxflow artifact download <IDENTITY> [DESTINATION] [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--force` | `-f` | Force creation if exists | `bool` | `false` |
