---
title: Artifact List
description: List artifacts in directory
---

### Usage

```bash [Terminal]
dxflow artifact list [DIRECTORY] [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--all` | `-a` | Include hidden artifacts | `bool` | `false` |
| `--depth` | `-d` | Depth of subdirectories to include | `int` | `0` |
| `--pattern` | `-p` | Pattern to filter artifacts | `string` | `` |
