---
title: Artifact
description: Manage files and directories in storage
---

### Usage

```bash [Terminal]
dxflow artifact
```

### Subcommands

| Command | Description | Help |
|---------|-------------|------|
| [**list**](/docs/cli/artifact/list) | List artifacts | `dxflow artifact list --help` |
| [**create**](/docs/cli/artifact/create) | Create artifact | `dxflow artifact create --help` |
| [**upload**](/docs/cli/artifact/upload) | Upload artifact | `dxflow artifact upload --help` |
| [**download**](/docs/cli/artifact/download) | Download artifact | `dxflow artifact download --help` |
| [**rename**](/docs/cli/artifact/rename) | Rename artifact | `dxflow artifact rename --help` |
| [**share**](/docs/cli/artifact/share) | Share artifact | `dxflow artifact share --help` |
| [**zip**](/docs/cli/artifact/zip) | Zip artifact | `dxflow artifact zip --help` |
| [**unzip**](/docs/cli/artifact/unzip) | Unzip artifact | `dxflow artifact unzip --help` |
| [**delete**](/docs/cli/artifact/delete) | Delete artifact | `dxflow artifact delete --help` |
