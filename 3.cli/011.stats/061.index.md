---
title: Stats
description: Display engine statistics and resource usage
---

### Usage

```bash [Terminal]
dxflow stats
```

### Output

```text [Terminal]
● Cpu
  User       12%  (8 CORE)
  System      4%
● Gpu
  Usage      78%  (2 GPU)
  Memory     75%  (18 GB / 24 GB)
● Memory
  Physical   46%  (31 GB)
  Swap        0%
● Disk
  Total      38%  (195 GB)
● Network
  Sent       1.2 GB
  Received   4.7 GB
```

The `Gpu` block renders directly below `Cpu` and is omitted entirely on hosts where no GPU is detected (Linux with `nvidia-smi`/`rocm-smi`).
