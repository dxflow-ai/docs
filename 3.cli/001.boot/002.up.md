---
title: Boot Up
description: Start engine with configured servers
---

### Usage

```bash [Terminal]
dxflow boot up [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--http` |  | Serve over http | `bool` | `false` |
| `--https` |  | Serve over https | `bool` | `false` |
| `--proxy` | `-p` | Setup proxy | `bool` | `false` |
| `--bridge` | `-b` | Setup bridge | `bool` | `false` |
| `--daemon` | `-d` | Run as daemon service | `bool` | `false` |
