---
title: Boot
description: Start and stop the dxflow engine
---

### Usage

```bash [Terminal]
dxflow boot
```

### Subcommands

| Command | Description | Help |
|---------|-------------|------|
| [**up**](/docs/cli/boot/up) | Start the engine | `dxflow boot up --help` |
| [**down**](/docs/cli/boot/down) | Stop the engine | `dxflow boot down --help` |
