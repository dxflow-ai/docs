---
title: Workflow
description: Manage workflow lifecycle, logs, and statistics
---

### Usage

```bash [Terminal]
dxflow workflow
```

### Subcommands

| Command | Description | Help |
|---------|-------------|------|
| [**list**](/docs/cli/workflow/list) | List workflows | `dxflow workflow list --help` |
| [**create**](/docs/cli/workflow/create) | Create workflow | `dxflow workflow create --help` |
| [**events**](/docs/cli/workflow/events) | Workflow events | `dxflow workflow events --help` |
| [**steps**](/docs/cli/workflow/steps) | Workflow steps | `dxflow workflow steps --help` |
| [**logs**](/docs/cli/workflow/logs) | View logs | `dxflow workflow logs --help` |
| [**stats**](/docs/cli/workflow/stats) | Workflow stats | `dxflow workflow stats --help` |
| [**start**](/docs/cli/workflow/start) | Start workflow | `dxflow workflow start --help` |
| [**stop**](/docs/cli/workflow/stop) | Stop workflow | `dxflow workflow stop --help` |
| [**remove**](/docs/cli/workflow/remove) | Delete workflow | `dxflow workflow remove --help` |
| [**prune**](/docs/cli/workflow/prune) | Prune workflows | `dxflow workflow prune --help` |
