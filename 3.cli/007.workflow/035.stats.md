---
title: Workflow Stats
description: Show workflow resource statistics
---

### Usage

```bash [Terminal]
dxflow workflow stats <IDENTITY>
```

### Output

```text [Terminal]
cpu 41% · mem 6.2/8G
```

A live, self-updating line aggregated across the workflow's running steps — CPU relative to the workflow's core limit, memory used / limit. It refreshes every 2s until interrupted; only started workflows report usage.
