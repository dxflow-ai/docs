---
title: Engine ping request
description: Request of ping engine
---

### Endpoint

```bash
GET /api/engine/ping/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default | Range |
|-----|---------|-------|
| interval | `1000` | `1000`–`30000` (clamped server-side) |
| count | `3` | `1`–`300` (clamped server-side) |

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": 0
    }
]
```
