title: Engine ping request
description: Request of ping engine
---

### Endpoint

```bash
GET /api/engine/ping/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default |
|-----|---------|
| interval | `1000` |
| count | `3` |

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": 0
    }
]
```
