---
title: Engine stat find gpu request
description: Request of gpu engine stat find
---

### Endpoint

```bash
GET /api/engine/stat/find/gpu/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default |
|-----|---------|
| day |  |
| hour |  |
| month |  |
| year |  |

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": {
            "points": {
                "gpu0": {
                    "utilization": 0,
                    "memory_used": 0,
                    "memory_total": 0,
                    "temperature": 0,
                    "power": 0
                }
            },
            "time": ""
        }
    }
]
```

### Notes

- Master Permission Required
- Collected on Linux only, via `nvidia-smi` (NVIDIA) or `rocm-smi` (AMD); empty when no GPU is detected.
