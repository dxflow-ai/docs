---
title: Artifact zip request
description: Request of zip artifact
---

### Endpoint

```bash
PUT /api/artifact/zip/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "identity": "",
    "quiet": false
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": {
            "destination": ""
        }
    }
]
```

### Notes

- Artifact Permission Required
