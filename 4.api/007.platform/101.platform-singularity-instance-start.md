---
title: Platform singularity instance start request
description: Request of start platform singularity instance
---

### Endpoint

```bash
POST /api/platform/singularity/instance/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "bind": [],
    "cpu": "",
    "env": [],
    "fakeroot": false,
    "gpu": "",
    "identity": "",
    "image": "",
    "memory": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    },
    {
        "kind": "entity",
        "payload": {
            "img": "",
            "instance": "",
            "ip": "",
            "log_err_path": "",
            "log_out_path": "",
            "pid": 0
        }
    }
]
```

### Notes

- Platform Permission Required
- Read-Only Permission Restricted
