---
title: Platform docker container start request
description: Request of start platform docker container
---

### Endpoint

```bash
POST /api/platform/docker/container/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "cpu": "",
    "env": [],
    "gpu": "",
    "identity": "",
    "image": "",
    "memory": "",
    "port": [],
    "volume": []
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    },
    {
        "kind": "entity",
        "payload": {
            "command": "",
            "created_at": "",
            "id": "",
            "image": "",

            "names": "",
            "networks": "",

            "ports": "",
            "running_for": "",
            "size": "",
            "state": "",
            "status": ""
        }
    }
]
```

### Notes

- Platform Permission Required
- Read-Only Permission Restricted
