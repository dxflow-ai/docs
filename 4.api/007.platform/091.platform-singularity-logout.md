---
title: Platform singularity logout request
description: Request of logout platform singularity
---

### Endpoint

```bash
PUT /api/platform/singularity/logout/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    }
]
```

### Notes

- Platform Permission Required
