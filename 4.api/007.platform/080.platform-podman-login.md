---
title: Platform podman login request
description: Request of login platform podman
---

### Endpoint

```bash
PUT /api/platform/podman/login/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "password": "",
    "username": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    }
]
```

### Notes

- Platform Permission Required
