---
title: Auth verify request
description: Request of verify auth
---

### Endpoint

```bash
POST /api/auth/verify/
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "identity": "",
    "lifetime": "",
    "signature": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": {
            "expires_at": 0,
            "permissions": [
                ""
            ],
            "token": ""
        }
    }
]
```

### Notes

- Rate Limited
- `lifetime` is optional (e.g. `"1h"`). Accepted range is 1 minute to 24 hours; out-of-range or omitted values fall back to the 1 hour default.
