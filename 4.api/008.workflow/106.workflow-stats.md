---
title: Workflow stats request
description: Request of workflow resource stats
---

### Endpoint

```bash
GET /api/workflow/stats/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default |
|-----|---------|
| identity |  |

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    }
]
```

### Notes

- Workflow Permission Required
- Supports streaming via `?stream` or `Accept: application/stream+json`
