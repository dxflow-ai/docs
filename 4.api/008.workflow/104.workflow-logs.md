---
title: Workflow logs request
description: Request of logs workflow
---

### Endpoint

```bash
GET /api/workflow/logs/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default |
|-----|---------|
| count | `1` |
| identity |  |
| step | `-1` |
| stdout | `false` |
| stderr | `false` |

When neither `stdout` nor `stderr` is set, both streams are returned.

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": {
            "output": "",
            "prefix": ""
        }
    }
]
```

### Notes

- Workflow Permission Required
