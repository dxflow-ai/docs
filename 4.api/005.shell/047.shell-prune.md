---
title: Shell prune request
description: Request of prune shell
---

### Endpoint

```bash
DELETE /api/shell/prune/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    }
]
```

### Notes

- Shell Permission Required
- Read-Only Permission Restricted
- Removes every shell; each pruned shell identity is returned as an entity.
